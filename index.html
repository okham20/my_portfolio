The Architect Profile

Ingeniero en Comunicaciones y Electrónica con una trayectoria que abarca desde el silicio hasta la nube. Mi enfoque une la física de los circuitos con la lógica abstracta del software de alto nivel, posicionándome como un arquitecto capaz de entender y optimizar sistemas complejos en su totalidad.

En Ford Motor Company actúo como Code Owner y punto final de escalamiento para blockers críticos, definiendo los estándares de arquitectura de automatización que garantizan la coherencia global de las plataformas HMI.

Mi misión es construir sistemas donde cada línea de código tenga un propósito arquitectónico: eliminando redundancia, estandarizando patrones y mentoreando a los equipos hacia una base de código sólida y escalable.

⬡ Code Owner Responsibilities

▸ Salud y coherencia del repositorio principal de automatización
▸ Definición de principios OOD y clases base estandarizadas en Python
▸ Punto final de escalamiento para blockers críticos (<1h resolución)
▸ Mentoría y revisión de código de desarrolladores junior
▸ Integración OpenCV para validación visual de interfaces HMI
▸ Gestión de pipelines CI/CD en Jenkins para releases FNV
